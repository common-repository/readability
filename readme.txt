== Description ==

Plugin which lets you to show posts in stylish, legible pop-up window, like in Safari.

Please report bugs in comments [here]http://wolnaelekcja.pl/wp-readability/) (I speak English and Polish, so be polite and use on of them) and don't forget about donate if you really like this plugin. :)

== Installation ==

1. Upload folder named `readability` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
